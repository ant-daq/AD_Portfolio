EPW Weather File Analysis

An interactive dashboard to explore EPW weather data.

Approach

The dashboard empowers users to dynamically explore different weather variables. After uploading an EPW weather file or using one of the pre-loaded ones, the users can gain statistical insights for each variable and compare their distributions against other factors. Additionally, the dashboard calculates and presents outdoor thermal comfort through an intuitive heat map visualization.

What I did

  • Envisioned the dashboard, its final use and interactive elements
  • Developed the interactive charts using Bokeh
  • Integrated all charts into a Streamlit application
