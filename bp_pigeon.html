Pigeon - Grasshopper Plug-in

A tool that enables the parametric exploration of natural ventilation potential of buildings.

Approach

Pigeon is a tool that allows the quick simulation of natural ventilation rates between rooms within the Grasshopper parametric environment. It uses the CONTAM software as calculation engine.

What I did

  • Ideation of the tool
  • Analysed the requirements for the text inputs of the CONTAM software
  • Developed python components in grasshopper for the quick setup of the room and opening properties
  • Developed python components to enable the visualisation of the natural ventilation flow rates across rooms
  • Implemented the CONTAM input text file generation
  • Tested the software and released it on the Food4Rhno platform
