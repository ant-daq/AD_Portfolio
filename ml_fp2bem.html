Floorplan 2 BEM (WIP)

A tool that allows the automatic generation of a 3D Building Energy Model that can be used for thermal and energy analysis.

Approach

I make use of the object detection model "You Only Look Once" (Yolo v8) for the fast prediction of building objects locations on the uploaded image. Once the building elements are located and correctly classified, an optimisation algorithm aligns all points to allow the generation of a clean gbXML export. The tool is currently WIP.

What I did

  • Envisioned the tool and collected data for training
  • Developed Yolo workflow to enable training
  • Developed grid-search approach for optimisation
  • Currently developing an interface to upload images and extract gbXML.
